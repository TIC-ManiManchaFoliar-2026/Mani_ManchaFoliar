# Mani_ManchaFoliar
El presente proyecto consiste en el ajuste y entrenamiento de un modelo de Deep Learning basado en YOLOv8 para la detección automática de mancha foliar en hojas de maní. A partir de un dataset de imágenes previamente recolectadas y etiquetadas, se entrena un modelo de visión por computadora capaz de identificar áreas afectadas por la enfermedaded.

Este sistema está orientado a brindar apoyo al GAD parroquial de Casanga, facilitando la identificación temprana de enfermedades en cultivos, lo cual permite mejorar la toma de decisiones agrícolas y reducir pérdidas en la producción.
